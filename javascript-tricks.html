JavaScript Mini Tricks & Snippets

Quick JS tips, tricks & mini challenges. Copy & practice easily!


Variables & Constants

  • Declare variable: let x = 5;
  • Constant: const PI = 3.14;

Functions & Arrow Functions

  • Normal function: function greet(name){ return 'Hello '+name; }
  • Arrow function: const square = x => x*x;

Array Tricks

  • Reverse array: arr.reverse()
  • Filter array: arr.filter(x => x>5)
  • Map array: arr.map(x => x*2)

DOM Manipulation

  • Get element: document.getElementById('id')
  • Change text: element.textContent = 'Hello'
